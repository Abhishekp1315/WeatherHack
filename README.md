# Weather Dashboard

A real-time weather dashboard built with React.js and OpenWeather API. View weather by location with icons. Designed for desktop or home screen embedding.

## Features
- Search weather by city
- View current temperature, description, and icon
- Use geolocation to get weather for your current location
- Responsive, clean UI

## Setup
1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Get an OpenWeather API key:**
   - Sign up at [OpenWeather](https://openweathermap.org/api)
   - Copy your API key
3. **Configure the API key:**
   - Open `src/weatherApi.js`
   - Replace `'YOUR_OPENWEATHER_API_KEY'` with your actual API key

## Run the app
```bash
npm start
```

The app will be available at [http://localhost:3000](http://localhost:3000)

## Build for production
```bash
npm run build
```
